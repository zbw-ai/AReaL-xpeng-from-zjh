# Qwen3.5-35B-A3B 32K RLVR Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Enable RLVR training of Qwen3.5-35B-A3B (MoE, 64 experts, top-4, GDN hybrid attention) on AReaL with 32K sequence length using 32× A100-80GB GPUs.

**Architecture:** Two-phase approach. Phase 1 uses the existing Archon engine with FSDP-only parallelism (zero new model code) to validate the full RLVR pipeline end-to-end. Phase 2 adds TP+EP to Archon's Qwen3.5 parallelize (porting from the working Qwen3 implementation) for production-grade throughput. GPU layout: 16 training + 16 SGLang inference.

**Tech Stack:** Archon engine (FSDP2/DTensor), SGLang inference, fla (flash-linear-attention for GDN kernels), causal-conv1d, torch.compile

**Memory Budget (Phase 1, Archon FSDP-only, 16 training GPUs):**

| Component | Per GPU | Notes |
|-----------|---------|-------|
| Actor params (BF16, sharded/16) | 4.4 GB | 35.5B × 2 / 16 |
| Optimizer (FP32, sharded/16) | 26.6 GB | 35.5B × 12 / 16 |
| Gradients (FP32, sharded/16) | 8.9 GB | 35.5B × 4 / 16 |
| Ref params (BF16, sharded/16) | 4.4 GB | Forward-only, colocated |
| Activations (32K, with AC) | ~6 GB | 1 layer at a time + all-gather buffer |
| Framework overhead | ~5 GB | NCCL, CUDA context |
| **Total** | **~55 GB** | **69% of 80 GB — fits** |

**Memory Budget (Phase 2, TP=4 EP=4 DP=1, 16 training GPUs):**

| Component | Per GPU | Notes |
|-----------|---------|-------|
| Expert params (BF16, EP=4) | 7.5 GB | 30B / EP(4) × 2 |
| Non-expert params (BF16, TP=4) | 1.375 GB | 5.5B / TP(4) × 2 |
| Optimizer (FP32, no DP to shard) | 35.5 GB | (7.5+1.375)B × 4 × 3 — needs `offload_params` or DP≥2 |
| Ref + Activations + overhead | ~16 GB | |
| **Total (no offload)** | **~60 GB** | **75% — tight but feasible** |

> **Note:** With TP=4, EP=4, DP=16/(4×4)=1 — no data parallelism to shard optimizer. If memory is tight, use `archon:(attn:d2t2|ffn:d2e8)` (TP=2, EP=8, DP=2) which gives optimizer sharding across DP=2 and fits in ~44 GB. Trade-off: less TP means higher activation memory per GPU.

```python
def freeze_moe_router(model: torch.nn.Module) -> int:
    """Freeze MoE router weights to stabilize expert routing during RL training.

    Finds all parameters matching ``*.moe.router.*`` and sets
    ``requires_grad = False``.

    Args:
        model: The model whose MoE routers should be frozen.

    Returns:
        Number of router parameters frozen.
    """
    count = 0
    for name, param in model.named_parameters():
        if ".moe.router." in name:
            param.requires_grad_(False)
            count += 1
    return count
```

```python
# fuyao_examples/test_sglang_qwen3_5.py
"""Validate SGLang can serve Qwen3.5-35B-A3B with TP=4."""

import subprocess
import sys
import time

import requests

MODEL_PATH = "/dataset_rc_b1/models/Qwen3.5-35B-A3B"
TP_SIZE = 4
PORT = 30000


def main():
    # Start SGLang server
    cmd = [
        sys.executable, "-m", "sglang.launch_server",
        "--model-path", MODEL_PATH,
        "--tp", str(TP_SIZE),
        "--port", str(PORT),
        "--dtype", "bfloat16",
        "--context-length", "32768",
        "--mem-fraction-static", "0.82",
        "--disable-custom-all-reduce",
    ]
    print(f"Starting SGLang: {' '.join(cmd)}")
    proc = subprocess.Popen(cmd)

    # Wait for server
    for i in range(120):
        try:
            resp = requests.get(f"http://localhost:{PORT}/health", timeout=2)
            if resp.status_code == 200:
                print(f"Server ready after {i}s")
                break
        except Exception:
            pass
        time.sleep(1)
    else:
        proc.kill()
        raise RuntimeError("SGLang server failed to start in 120s")

    # Test generation
    payload = {
        "model": MODEL_PATH,
        "messages": [{"role": "user", "content": "What is 2+2? Think step by step."}],
        "max_tokens": 256,
        "temperature": 0.7,
    }
    resp = requests.post(
        f"http://localhost:{PORT}/v1/chat/completions",
        json=payload,
        timeout=60,
    )
    resp.raise_for_status()
    result = resp.json()
    content = result["choices"][0]["message"]["content"]
    print(f"Response ({len(content)} chars): {content[:200]}...")

    proc.kill()
    print("SGLang Qwen3.5 validation PASSED")


if __name__ == "__main__":
    main()
```

**Key TP plans by layer type:**

```python
def apply_non_moe_tp(
    model: nn.Module,
    tp_mesh: DeviceMesh,
    loss_parallel: bool = True,
) -> None:
    """Apply tensor parallelism to non-MoE components of Qwen3.5."""
    from torch.distributed.tensor import Replicate, Shard
    from torch.distributed.tensor.parallel import (
        ColwiseParallel, RowwiseParallel, SequenceParallel,
        PrepareModuleInput, parallelize_module,
    )
    from areal.models.parallel_styles import ReplicateParallel

    validate_qwen3_5_tp_constraints(model.model_args, tp_mesh.size())

    # Root-level modules (same as Qwen3)
    root_plan = {}
    if model.tok_embeddings is not None:
        root_plan["tok_embeddings"] = RowwiseParallel(
            input_layouts=Replicate(), output_layouts=Shard(1),
        )
    if model.norm is not None:
        root_plan["norm"] = SequenceParallel()
    if model.output is not None:
        root_plan["output"] = ColwiseParallel(
            input_layouts=Shard(1),
            output_layouts=Shard(-1) if loss_parallel else Replicate(),
            use_local_output=True,
        )
    if root_plan:
        parallelize_module(model, tp_mesh, root_plan)

    # Per-layer TP
    for layer_id, block in model.layers.items():
        layer_type = model.model_args.layer_types[int(layer_id)]

        if layer_type == "full_attention":
            # GatedAttention: wq is 2x width (query + gate), wk, wv, wo
            layer_plan = {
                "attention_norm": SequenceParallel(),
                "attention": PrepareModuleInput(
                    input_layouts=(Shard(1),),
                    desired_input_layouts=(Replicate(),),
                ),
                "attention.wq": ColwiseParallel(),      # [dim, 2*n_heads*head_dim] split on col
                "attention.wk": ColwiseParallel(),      # [dim, n_kv_heads*head_dim]
                "attention.wv": ColwiseParallel(),      # [dim, n_kv_heads*head_dim]
                "attention.wo": RowwiseParallel(output_layouts=Shard(1)),
                "attention.q_norm": ReplicateParallel(),  # per-head norm, replicate
                "attention.k_norm": ReplicateParallel(),
                "ffn_norm": SequenceParallel(),
            }
        else:
            # GatedDeltaNet: in_proj_qkv, in_proj_z, in_proj_a, in_proj_b, out_proj
            # conv1d groups are split across TP (depthwise, groups = channels)
            layer_plan = {
                "attention_norm": SequenceParallel(),
                "linear_attn": PrepareModuleInput(
                    input_layouts=(Shard(1),),
                    desired_input_layouts=(Replicate(),),
                ),
                "linear_attn.in_proj_qkv": ColwiseParallel(),   # [dim, key_dim*2+value_dim]
                "linear_attn.in_proj_z": ColwiseParallel(),      # [dim, value_dim]
                "linear_attn.out_proj": RowwiseParallel(output_layouts=Shard(1)),
                # A and B project to num_v_heads — split across TP
                "linear_attn.in_proj_a": ColwiseParallel(),      # [dim, num_v_heads]
                "linear_attn.in_proj_b": ColwiseParallel(),      # [dim, num_v_heads]
                # Bare params: A_log and dt_bias are per-head, shard across TP
                # (handled separately after parallelize_module)
                "ffn_norm": SequenceParallel(),
            }

        # Non-MoE FFN layers (if any dense layers exist)
        if hasattr(block, "feed_forward") and block.feed_forward is not None:
            layer_plan["feed_forward.w1"] = ColwiseParallel()
            layer_plan["feed_forward.w3"] = ColwiseParallel()
            layer_plan["feed_forward.w2"] = RowwiseParallel(output_layouts=Shard(1))

        parallelize_module(block, tp_mesh, layer_plan)

        # Handle conv1d and bare params for GDN layers
        if layer_type == "linear_attention" and hasattr(block, "linear_attn"):
            _shard_gdn_special_params(block.linear_attn, tp_mesh)
```

- [ ] **Step 3: Handle GDN conv1d and bare params**

```python
def _shard_gdn_special_params(gdn_module: nn.Module, tp_mesh: DeviceMesh) -> None:
    """Shard GDN-specific parameters that can't use standard TP plans.

    - conv1d: depthwise conv with groups=channels. Split groups across TP.
    - A_log, dt_bias: per-head params. Split across TP.
    - norm.weight: per-head norm. Split across TP.
    """
    from torch.distributed.tensor import distribute_tensor, Shard

    tp_size = tp_mesh.size()
    tp_rank = tp_mesh.get_local_rank()

    # conv1d weight: [channels, 1, kernel_size], split channels across TP
    if hasattr(gdn_module, "conv1d") and gdn_module.conv1d is not None:
        conv = gdn_module.conv1d
        # Slice the conv weight and groups
        total_channels = conv.weight.shape[0]
        local_channels = total_channels // tp_size
        start = tp_rank * local_channels
        end = start + local_channels
        conv.weight = nn.Parameter(conv.weight.data[start:end].contiguous())
        conv.in_channels = local_channels
        conv.out_channels = local_channels
        conv.groups = local_channels

    # Bare params: A_log, dt_bias — per-head, shard across TP
    for param_name in ("A_log", "dt_bias"):
        if hasattr(gdn_module, param_name):
            param = getattr(gdn_module, param_name)
            total_heads = param.shape[0]
            local_heads = total_heads // tp_size
            start = tp_rank * local_heads
            end = start + local_heads
            setattr(gdn_module, param_name, nn.Parameter(param.data[start:end].contiguous()))

    # norm.weight: per-head dim, shard across TP
    if hasattr(gdn_module, "norm") and hasattr(gdn_module.norm, "weight"):
        w = gdn_module.norm.weight
        total_dim = w.shape[0]
        local_dim = total_dim // tp_size
        start = tp_rank * local_dim
        end = start + local_dim
        gdn_module.norm.weight = nn.Parameter(w.data[start:end].contiguous())
```
